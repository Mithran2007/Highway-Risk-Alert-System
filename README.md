# 🚦 NH44 Highway Risk Alert System

A modern web-based Highway Risk Alert System that monitors highway incidents, road hazards, traffic conditions, and risk levels through an interactive dashboard.

---

## 📌 Project Overview

The NH44 Highway Risk Alert System is a front-end web application developed to simulate a real-time highway monitoring system. It provides users with live traffic alerts, accident reports, risk analytics, and highway status updates in a clean and responsive interface.

---

## ✨ Features

- 🚨 Live Highway Incident Alerts
- 🗺️ Interactive Highway Map
- 📊 Risk Analytics Dashboard
- 📈 Incident Statistics
- 📋 Active Incident Table
- 📢 Live Alert Feed
- 📱 Fully Responsive Design
- 🎨 Modern Dark Theme UI
- 📊 Interactive Charts using Chart.js
- ⚡ Smooth Animations
- 📞 Alert Subscription Form

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
